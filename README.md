## React Movies Applicaction

#### What this app can do?

- Search for movies
- View movies details
- View movies trailer
- Save movies as favorites or watched

#### Features

- Simple and beautiful
- Responsive
- Toggle theme ( dark and light mode )

#### Technologies

- HTML5
- CSS3
- TailwindCSS
- NodeJS
- RactJS
- localStorage
